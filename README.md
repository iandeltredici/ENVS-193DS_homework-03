# ENVS-193DS_homework-03
Homework set for ENVS 193DS Winter 2026

## General Information

This repo is for homework set #3 of the ENV S 193 Data Science class at UCSB.

This is my personal repo and I am the sole contributor.

All code files are in the 'code' folder. Images are also in the 'code' folder while data sets are in the 'data' folder. There is a .qmd file that contains models and tests for two data sets (salinity-pickleweed.csv and time_spent_reading.csv). Images of a personal data project and a visualization are also included.

The files include code for linear models, correlation tests, checking model assumptions, captions of each figure, written statements about models and results, and a statistical critique.
